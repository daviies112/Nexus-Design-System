## Opção 4: Web Scraping / Automação Google (n8n)

### Conceito
Automatizar coleta de dados públicos de empresas de semijoias.

### Workflow n8n

```javascript
WORKFLOW: "Coletar Empresas Semijoias"

1. HTTP Request → Google Custom Search API
   ├─ Query: "empresa semijoias" + cidade
   ├─ Filtro: site:.com.br OR site:.br
   └─ Retorna: URLs de empresas

2. Loop por cada URL:
   ├─ HTTP Request → Acessar site
   ├─ HTML Extract:
   │   ├─ Nome empresa (regex/xpath)
   │   ├─ Telefone (regex: \(\d{2}\) \d{4,5}-\d{4})
   │   ├─ Email (regex: .+@.+\..+)
   │   ├─ Endereço
   │   └─ Instagram/Facebook link
   └─ Salvar no Supabase

3. Instagram Graph API (opcional):
   ├─ Buscar @empresa no Instagram
   ├─ Verificar: Seguidores, posts, ativo?
   └─ Salvar dados

4. Validação WhatsApp:
   ├─ Evolution API
   ├─ Verificar se número está no WhatsApp
   └─ Atualizar base (whatsapp_ativo: true/false)

5. Enriquecimento final:
   ├─ Buscar CNPJ na Receita Federal (API pública)
   ├─ Classificar por porte
   └─ Marcar como "pronto para contato"
```

### Fontes para Scraping

| Fonte | O que coletar | Quantidade esperada |
|-------|---------------|---------------------|
| **Google Search** | Sites de empresas | 500-1.000 empresas |
| **Instagram** | Perfis @semijoias | 1.000-2.000 empresas |
| **Facebook Pages** | Páginas empresas | 500-1.000 empresas |
| **Reclame Aqui** | Empresas do setor | 200-500 empresas |
| **GuiaMais/TeleListas** | Dados comerciais | 300-600 empresas |

### Exemplo Prático n8n

```
FLUXO SIMPLES:
1. Schedule Trigger (roda 1x/dia)
   ↓
2. Google Custom Search
   Query: "semijoias São Paulo" site:.com.br
   Retorna: 10 URLs
   ↓
3. Loop URLs:
   HTTP Request → HTML Extract
   Regex telefone: \(\d{2}\) \d{4,5}-\d{4}
   Regex email: [a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,}
   ↓
4. Supabase Insert
   Salva: url, nome, telefone, email, cidade
   ↓
5. Wait 1 day → Repete com próxima cidade
```

**Em 30 dias:**
- 30 cidades × 10 empresas/dia = 300 empresas
- Custo: R$ 0 (APIs gratuitas)

### Vantagens
- ✅ Totalmente automatizado
- ✅ Custo zero (APIs gratuitas)
- ✅ Dados públicos (legal)
- ✅ Pode rodar continuamente
- ✅ Base sempre atualizada

### ⚠️ Atenção Legal
- ✅ Dados públicos: OK
- ✅ Para uso comercial próprio: OK
- ❌ Revenda de dados: NÃO
- ❌ SPAM: NÃO (seguir LGPD)

### Exemplo de classificação automática

```javascript
// Lógica de classificação
const classificarEmpresa = (cnaePrincipal, cnaeSecundarios) => {
  const tipos = [];
  
  // Verificar fabricação
  if (['3213-1/00', '2599-3/04'].includes(cnaePrincipal)) {
    tipos.push('FABRICANTE');
  }
  
  // Verificar banho/galvanoplastia
  if (cnaePrincipal === '2599-3/05' || 
      cnaeSecundarios?.includes('2599-3/05')) {
    tipos.push('BANHO');
  }
  
  // Verificar revenda
  if (cnaePrincipal === '4782-2/02') {
    tipos.push('REVENDEDORA');
  }
  
  return tipos.join('+') || 'OUTRO';
};
```

Vantagem: receita garantida, independente de indicação

### 2. Indicação obrigatória (crescimento exponencial)

**Como funciona:**

```
Fabricante se cadastra → GRATUITO
   ↓
Condição: "Você indica suas revendedoras para usar nosso sistema"
   ↓
Fabricante indica 10 revendedoras
   ↓
10 revendedoras entram no sistema
   ↓
Você ganha: 10 × R$ 2.830/mês = R$ 28.300/mês
   ↓
Ciclo se repete: mais revendedoras = mais fabricantes querem entrar
```

Cálculo:

```
50 fabricantes × 10 revendedoras indicadas = 500 revendedoras
   ↓
Mas muitas revendedoras são compartilhadas entre fabricantes
   ↓
Estimativa realista: 250-300 revendedoras únicas
   ↓
Receita: 250 × R$ 2.830/mês = R$ 707.500/mês
   ↓
Receita anual: R$ 8.490.000/ano
```

**Como funciona:**

```
Quanto mais fabricantes:
   ↓

   ↓
Mais revendedoras querem usar sistema
   ↓
Mais fabricantes querem entrar (acesso a clientes)
   ↓
CICLO VIRTUOSO
```
